Guide d'utilisation

×

Suivez ces étapes pour bien démarrer avec JIRAMA Charge Manager.

1

Ajouter des colocataires

Allez dans l'onglet Colocataires et cliquez sur Ajouter un colocataire. Renseignez son nom, email, téléphone et son coefficient de présence (1 = présent tout le mois).

2

Ajouter des appareils électriques

Dans l'onglet Appareils, ajoutez chaque appareil avec sa puissance (W), ses heures d'utilisation par jour et le nombre de jours par mois. Indiquez s'il est individuel ou partagé.

3

Saisir les factures JIRAMA

Dans Paramètres → JCM Calculate, entrez le montant total de vos factures d'électricité et d'eau ainsi que les consommations. Le prix moyen par kWh et par m³ sera calculé automatiquement.

4

Visualiser la facturation

L'onglet Facturation calcule automatiquement la répartition des charges entre tous les colocataires selon leur coefficient de présence et leurs appareils.

5

Générer une facture PDF

Dans l'onglet Facturation, cliquez sur Générer PDF pour obtenir une facture professionnelle téléchargeable. Vous pouvez aussi partager via WhatsApp ou copier dans le presse-papier.

6

Suivre l'historique

L'onglet Historique conserve toutes les factures générées et les paiements enregistrés. Marquez les paiements comme reçus directement dans l'onglet Facturation.

7

Sauvegarder vos données

Utilisez Paramètres → Cloud Backup pour connecter votre Google Drive et sauvegarder automatiquement. Vous pouvez aussi exporter en JSON ou Excel depuis le Dashboard.

8

Scanner une facture

Cliquez sur Scanner facture dans le Dashboard pour importer automatiquement les montants depuis une photo de votre facture JIRAMA.

Astuces

Le coefficient de présence permet d'ajuster la part de chaque colocataire absent pendant une partie du mois (ex: 0.5 = 15 jours présent).
Les appareils partagés (frigo, TV) sont automatiquement répartis entre tous selon leur coefficient.
Activez les tranches tarifaires personnalisées pour utiliser les tarifs exacts de votre contrat JIRAMA.
Le mode invité permet d'explorer l'application sans créer de compte.

Foire aux questions

×
Comment fonctionne le coefficient de présence ?

Le coefficient permet d'ajuster la part des charges selon la présence réelle. Exemple : un colocataire présent 50% du mois aura un coefficient de 0.5. Cela permet de ne pas payer pour les périodes d'absence.

Différence entre appareil individuel et partagé ?

Appareil individuel : Facturé uniquement à la personne qui l'utilise (ex: ordinateur personnel).
Appareil partagé : Réparti équitablement entre tous les colocataires selon leur coefficient de présence (ex: réfrigérateur, télévision).

Comment importer mes factures JIRAMA ?

Rendez-vous dans l'onglet "Paramètres" et saisissez manuellement le montant de vos factures d'électricité et d'eau. L'application calculera automatiquement la répartition entre les colocataires.

Puis-je exporter mes données ?

Oui ! Utilisez le bouton "Exporter" dans le Dashboard pour sauvegarder toutes vos données au format JSON. Vous pourrez les réimporter plus tard avec le bouton "Importer".

Comment générer une facture PDF ?

Dans l'onglet "Facturation", cliquez sur "Générer PDF". Une facture professionnelle sera automatiquement téléchargée avec le détail des charges par personne.

Les données sont-elles sauvegardées ?

Oui ! Toutes vos données sont automatiquement sauvegardées dans votre navigateur (localStorage). Vous pouvez aussi les exporter pour les conserver en sécurité.
